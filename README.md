# What is an arrow function?
Arrow Function হলো জাভাস্ক্রিপ্টে ফাংশন লেখার একটি আধুনিক, সংক্ষিপ্ত এবং সহজ পদ্ধতি 

# What are template literals?
Template Literals হলো জাভাস্ক্রিপ্টে স্ট্রিং (String) লেখার একটি আধুনিক এবং শক্তিশালী পদ্ধতি, যা ES6 ভার্সনে যুক্ত করা হয়েছে। সাধারণ স্ট্রিং-এ আমরা যেখানে কোটেশন (' বা ") ব্যবহার করি, টেমপ্লেট লিটারেলসে সেখানে ব্যাকটিক (`) ব্যবহার করা হয়।
